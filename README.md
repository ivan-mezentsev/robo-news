# Robo News

A small collection of Rust tools that turn news sources into publishable content.

## What’s inside

- `downloader-feed1` — downloads source pages.
- `scraper` — extracts readable article content from downloaded pages.
- `parser-feed1` — turns a source feed into structured items.
- `translator` — translates prepared text.
- `publisher` — publishes the final output.

## Repository layout

This repository contains a small set of microservices.
